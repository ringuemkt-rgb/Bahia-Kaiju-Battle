# Colossos do Baixo Sul — Production Kit

Este diretório adiciona ao repositório uma atualização **não destrutiva** com a visão mais recente do projeto, organizada para orientar a evolução de **Bahia-Kaiju-Battle** para um jogo mais denso, regional e profissional.

## O que foi adicionado
- visão mestra do projeto
- bíblia do mundo do Baixo Sul da Bahia
- arsenal monstruoso, customização e bosses
- sistemas vivos de IA, clima, maré e eventos
- vertical slice recomendado
- templates de dados em JSON
- scaffold técnico inicial para cenas e systems

## Direção criativa consolidada
O projeto passa a priorizar:
- **Ituberá** como ponto de entrada do mundo
- **Pancada Grande**, **Itacaré**, **Baía de Camamu**, **Ilha do Timbuca**, **Morro de São Paulo**, **Valença** e **Salvador** como regiões-chave
- monstros originais inspirados em ecossistemas do Baixo Sul
- destruição sistêmica com reação militar e civil
- mundo vivo com wanted militar, clima, maré e IA de NPCs
- uso de barcos, lanchas, estruturas e props costeiros como arma

## Observação importante
Este kit não substitui sozinho a implementação final do jogo. Ele atualiza o repositório com uma base forte de pré-produção + dados + scaffold, sem quebrar o código atual.
